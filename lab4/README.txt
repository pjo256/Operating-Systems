
In this directory, you will find a linked list implementation of a queue (doubly-linked list) and the page manager code, containing the specified page replacement algorithms.

	i)	Compilation command: gcc pager.c queue.c -o pager
	ii)	Running command (takes at most two command line arguments)
			Unix -> ./pager <machine_sz> <page_sz> <process_sz> <job_mix> <num_references> <replacement_algo> [debug_flag]
			MSDOS -> pager <machine_sz> <page_sz> <process_sz> <job_mix> <num_references> <replacement_algo> [debug_flag]
